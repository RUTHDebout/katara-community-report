# Security Policy

## Reporting vulnerabilities

If you discover a security vulnerability, please report it responsibly by opening a private communication channel with the maintainer before public disclosure.

Please include:

- Vulnerability description
- Steps to reproduce
- Potential impact
- Suggested mitigation if available

## Scope

This project aims to follow secure development practices, especially for:

- User-submitted reports
- Geolocation data
- Authentication workflows
- API integrations
- Community-generated content
