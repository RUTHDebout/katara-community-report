# Deployment Guide

## Frontend

Recommended platforms:
- Vercel
- Netlify

Development:

```bash
npm install
npm run dev
```

## Backend

Recommended platforms:
- Railway
- Render
- Fly.io

Development:

```bash
pip install -r requirements.txt
uvicorn main:app --reload
```

## Future integrations

- SMS gateways
