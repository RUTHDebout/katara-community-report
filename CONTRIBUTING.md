# Contributing to KATARA Community Report

Thank you for your interest in contributing.

## Ways to contribute

- Documentation improvements
- Frontend and UI/UX
- Geospatial workflows
- Accessibility improvements
- Translation and multilingual support
- Backend development
- AI-assisted classification workflows
- Civic-tech deployment guidance

## Development principles

- Accessibility first
- Low-bandwidth compatibility
- Open standards
- Community-centered design
- Ethical and privacy-aware implementation

## Pull requests

1. Fork the repository.
2. Create a feature branch.
3. Make clear and documented commits.
4. Submit a pull request with explanation.

## Reporting issues

Please provide:

- Context
- Reproduction steps
- Expected behavior
- Screenshots/logs if applicable

## Community values

We encourage respectful collaboration and inclusive participation.
